# Stack Overflow Developer Survey 2024 — анализ и визуализация

Курсовая работа: исследование факторов, влияющих на компенсацию IT-специалистов на глобальном рынке труда, на основе данных Stack Overflow Annual Developer Survey 2024. Результат — интерактивный дашборд в Yandex DataLens.

## Интерактивный дашборд

**Публичная ссылка:** https://datalens.yandex/8nv6m7mu0cd0s

Дашборд содержит:
- три KPI-карточки: общее число респондентов, медианный годовой доход (USD), доля разработчиков, использующих AI-инструменты;
- горизонтальная гистограмма медианного дохода по странам (топ-стран по числу респондентов);
- сводная таблица-«тепловая карта»: язык программирования × группа профессионального стажа → медианный годовой доход;
- гистограмма медианного дохода по группам стажа работы (от 00–01 до 30+ лет);
- гистограмма популярности языков программирования по числу пользователей;
- агрегированная таблица: страна × язык × стаж с медианным доходом и числом респондентов в каждом срезе;
- три глобальных селектора: страна, тип разработчика, группа стажа работы.

## Цель и задачи работы

**Цель:** выявить ключевые факторы ценообразования на глобальном рынке труда в IT-сфере — оценить влияние технологического стека, профессионального опыта и географии на уровень компенсации разработчиков.

## Данные

| Параметр | Значение |
|---|---|
| Источник | [Stack Overflow Annual Developer Survey 2024](https://survey.stackoverflow.co/) |
| Объём | 65 437 респондентов, 114 признаков |
| Период сбора | май 2024 года |
| Тип лицензии | Open Database License (ODbL) |
| Ключевая целевая переменная | `ConvertedCompYearly` — годовой доход в USD |

Исходный файл `survey_results_public.csv` в репозиторий **не входит** (см. `.gitignore`).
